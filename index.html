プログラム復習コンテンツ

keyCode変数

サンプルコードの結果

サンプルコード

　　 int x=215;

　

　　 void setup(){

　　 　size(480, 120);　　

　　 }

　　

　　 void draw(){

　　 　if(keyPressed && (key == CODED)){

　　 　　if(keyCode==LEFT){

　　 　　　x--;

　　 　　}else if(keyCode==RIGHT){

　　 　　　x++;

　　 　　}

　　 　}

　　 　rect(x, 45, 50, 50);

　　 }

説明文

　keyCode関数は、key関数で判別することのできないボタンを判別するときに使う。

　主に、（矢印ボタン、Alt、Ctrl、Shift）を使用する際に用いる。

　　矢印ボタンは（UP, DOWN, LEFT, RIGHT）

　　Altボタンは（ALT）

　　Ctrlボタンは（CONTROL）

　　Shiftボタンは（SHIFT）

　これらはコード化されたボタンと呼ばれ、使用する際に（key==CODED）と条件を付けることで、押されたボタンはコード化されたものか判別する必要がある。

　使いかたは例の通りである。

　例では、→ボタンを押すと四角形が右に移動し、←に押すと左に移動する仕組みである。
